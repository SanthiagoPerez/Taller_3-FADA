## EJEMPLO

Arreglo: [6, 1, 9, 6, 9, 6, 7, 3, 3, 6] 
Moda(s): [6]

Este seria el ejemplo de la ejecución de un arreglo de 10 elementos, 
donde se ve evidenciado que el número que más se repite es el 6, por ende, es la moda.
